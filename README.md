# README
# パン屋訪問記録アプリ「Pantastic」

## 概要
行ったことのあるパン屋さんの記録管理に、また他のユーザーの投稿から次に行きたいパン屋さんの候補探しに使っていただけます。

## コンセプト
パン屋巡りの便利ツール。  
覚えにくいパン屋さんの名前もアプリで簡単管理。

## バージョン
ruby 2.3.0  Rails 5.1.6

## 機能一覧
+ ログイン機能
+ ユーザー登録機能  
    + 名前、メールアドレス、アイコン、パスワ
    ード必須  
+ 投稿一覧表示機能 
    + 一覧画面でお店名検索機能
+ 投稿機能
    + お店の名前、場所、外観写真、パンの写真と名前が投稿できる
    + 編集、削除は投稿者のみ可能

+ お気に入り機能（行ってみたい！機能）
    + 投稿者は自分のブログにお気に入りできない

+ お気に入り一覧機能
+ ユーザー編集機能
